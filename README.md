# Moto A1/A2 Tests

Webapp estática para practicar tests de permiso A1/A2.

## Uso local

```bash
python3 -m http.server 8000
```

Abre `http://127.0.0.1:8000/index.html`.

## Datos

- `tests.json`: preguntas, respuestas, explicaciones e IDs internos.
- `images/`: imágenes locales referenciadas por el JSON.
